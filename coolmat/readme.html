This is a custom theme developed from the Underscores theme by Automattic. It is created to provide a single-page store template for cool mat, a Korean food brand based in Seoul and expanding into other franchise markets. PHP has been written to interface with the Wordpress backend and allow for consistent formatting with new posts. The user can change menu items and the page will still function correctly as a menu link. CSS grid is used for modern, sleek design and responsive design allows this site to look great on mobile, as this site will be mainly used on device by customers.
